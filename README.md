# iia-ia-bomberman
Bomberman agent for AI subject

## How to install

Make sure you are running Python 3.5.

`$ pip install -r requirements.txt`

*Tip: you might want to create a virtualenv first*

## How to play

Open 3 terminals:

`$ python3 server.py`

`$ python3 viewer.py`

`$ python3 student.py`
